# Model Tournament

**Date:** 2026-04-12
**Status:** Automated

New models appear on HuggingFace every week. The question isn't "is it good?" — it's "is it better than what we have, at the specific thing each Jedi needs?"

The Model Tournament answers that question automatically. No human evaluation. No vibes. Just benchmarks, per-category comparison, and a commit to `champions.json` if the challenger wins.

## The Pipeline

```
Model Scout (weekly LaunchAgent)
    │ discovers candidate on HuggingFace
    ▼
Model Tournament (model_tournament.py)
    │
    ├── Serve candidate on temp port (:9876)
    ├── Carmack v2 — 40 infrastructure tasks
    ├── Coding bench — 15 real-world tasks
    │
    ├── Per-category comparison against champions.json
    │
    ├── WIN (>5% improvement) → update routing + notify
    └── LOSE → log results, champions unchanged
```

```bash
# Evaluate any model with one command
python model_tournament.py \
  --model ./models/NewModel-4bit \
  --label "NewModel" \
  --apply --notify
```

## Jedi Assignment

The tournament decides which model serves which agent. Three tiers:

| Tier | Agents | Model | Why |
|------|--------|-------|-----|
| **Cloud** | Windu, Mothma, Jocasta | Opus 4.6 | Best overall (0.845), tool precision (1.0) |
| **Local Ops** | Yoda, Qui-Gon, Ahsoka | Coder-14B | Best correlation (0.917), satellite (1.0), free |
| **Local Secure** | Cilghal, Mundi | Gemma4+LoRA | Best jailbreak (0.850), health/fund data stays local |

<Aside type="note">
Cilghal and Mundi MUST stay on the local secure tier regardless of benchmark scores. Health data and fund terms cannot leave the machine. Privacy is a hard constraint, not a performance tradeoff.
</Aside>
